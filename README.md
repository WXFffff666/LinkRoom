# LinkRoom

Windows 便携式 P2P 联机工具。输入房间号+密码，一键直连。

## 功能

- **零配置** — 无需账号/注册/服务器
- **P2P 优先** — 自动检测 NAT 类型，选择最优穿透策略
- **简洁 UI** — WPF 界面，展开高级设置可自定义共享节点/STUN/UPnP
- **安全** — 房间密码不保存、不通过命令行传递、日志自动脱敏

## 系统要求

- Windows 10 / 11 x64
- 首次运行需管理员权限（安装虚拟网卡驱动 Wintun）

## 快速开始

1. 下载 `LinkRoom-win-x64.zip`
2. 解压得到 `LinkRoom.exe`
3. 双击运行 → 输入房间号+密码 → 点击"连接 (Connect)"

## 高级设置

| 选项 | 说明 | 默认 |
|------|------|------|
| 共享节点 | 通过中继节点连接（非 P2P） | 关闭 |
| 禁用 UPnP | 不自动配置路由器端口映射 | 启用 |
| 自定义 STUN 服务器 | 自建 STUN 服务器地址 | Google STUN |
| 最大重连次数 | 掉线后自动重试次数 | 5 |
| 静态虚拟 IP | 指定固定虚拟 IP（不推荐） | DHCP |

## 安全与隐私

- 房间密码**不会**保存到磁盘，每次需重新输入
- 密码**不会**出现在进程命令行参数中
- 日志默认对密码和公网 IP 脱敏
- 共享节点会看到你的连接元数据，仅在你明确启用的前提下使用
- Wintun 驱动安装需管理员权限（UAC 弹窗为正常现象）

## 卸载

1. 关闭 LinkRoom
2. 运行 `tools\cleanup.ps1`（管理员权限）
3. 或手动删除：`%LOCALAPPDATA%\LinkRoom\`
